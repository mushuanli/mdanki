
### **第五章：抛体运动**
> **核心思想**：将复杂的曲线运动分解为两个相互独立的简单直线运动来研究。
#### **1. 曲线运动**
- **定义**：
  - **曲线运动**：轨迹是曲线的运动。
  - **切线**：当割线上的两点无限接近时，该割线称为曲线在该点的切线。
- **特性**：
  - 速度方向沿轨迹的切线方向（图5.1-2）。
  - 是变速运动（速度方向时刻变化，加速度≠0）。
- **条件**：
- **运动条件**：物体所受**合外力 (F) 的方向**与**速度 (v) 的方向**不在同一直线上。

> 🔑 **教师提示**：合力与速度是否共线，是判断物体做直线运动还是曲线运动的根本依据。
> - **共线** → 直线运动
> - **不共线** → 曲线运动

#### **2. 运动的合成与分解**
    *   **核心思想**：将复杂的曲线运动分解为两个或多个简单的直线运动来研究。这是物理学中处理复杂问题的重要思想。
    *   **基本法则**：位移、速度、加速度的合成与分解都遵循**矢量运算法则（平行四边形定则）**。
    *   **关键特性（分运动的独立性）**：各个分运动独立进行，互不干扰。一个方向的运动不影响另一个方向的运动。
- **定义**：
  - **合运动**：物体实际运动。
  - **分运动**：将合运动分解得到的独立运动。
  - **运动的合成**：由分运动求合运动。
  - **运动的分解**：由合运动求分运动。
- **核心规律**：
  - 遵循矢量运算法则（平行四边形定则）。
  - **分运动的独立性**：各分运动相互独立，互不影响。
- **公式**：
  - 位移分解：`x = vₓ t`, `y = vᵧ t`（匀速分运动）
  - 速度合成：`v = √(vₓ² + vᵧ²)`, `tan θ = vᵧ / vₓ`（θ为合速度与x轴夹角）
- **实验验证**：蜡块实验（图5.2-1）证明合运动轨迹为直线。

#### **3. 实验：探究平抛运动的特点**
- **定义**：
  - **抛体运动**：只受重力，初速度不为零的运动。
  - **平抛运动**：初速度水平的抛体运动。
- **实验方法**：
  - **频闪照相法**（图5.3-1）：分析水平匀速、竖直自由落体。
  - **对比实验**（图5.3-2）：A球平抛与B球自由落体同时落地。
- **结论**：
  - 水平方向：匀速直线运动（`vₓ = v₀`）。
  - 竖直方向：自由落体运动（`vᵧ = gt`, `y = ½gt²`）。

#### **4. 抛体运动的规律**
- **性质**：平抛是匀变速曲线运动（`a = g`，竖直向下）。
- **公式**：
  - 速度：
    - 水平：`vₓ = v₀`
    - 竖直：`vᵧ = gt`
    - 合速度大小：`v = √(v₀² + (gt)²)`
    - 速度方向与水平夹角θ：`tan θ = vᵧ / vₓ = gt / v₀`
  - 位移：
    - 水平：`x = v₀ t`
    - 竖直：`y = ½gt²`
  - 轨迹方程（消去时间t）：`y = (g/(2v₀²)) x²`（抛物线）。
            *   **教师提示**：这个方程表明平抛运动的轨迹是一条**抛物线**。
- **斜抛处理**：分解为水平匀速（`v₀ₓ = v₀ cos θ`）和竖直上抛（`v₀ᵧ = v₀ sin θ`）。

### **第六章 圆周运动**
> **核心概念**：用线速度、角速度等描述运动，并用力（向心力）解释其原因。
#### **核心概念**：描述圆周运动的物理量及向心力
#### **1. 基本概念**
- **圆周运动**：轨迹为圆或圆弧的运动。
- **线速度 (v)**：描述沿圆弧运动的快慢。
  - 定义：`v = Δs / Δt`，方向沿切线。
- **角速度 (ω)**：描述绕圆心转动的快慢
  - 定义：`ω = Δθ / Δt`，单位 rad/s。
- **周期 (T)** (运动一周的时间。) 与转速 (n)：
  - **核心关系式**：`v = ωr`，`T = 2π/ω`，`v = 2πr/T`。
- **匀速圆周运动**：：线速度**大小**（速率）和角速度都恒定的圆周运动。但速度方向在变，所以仍是**变速运动**。
- **向心力 (Fₙ)**：
  - 定义：指向圆心的合力（效果力，由重力/弹力等提供）。
    *   **教师提示**：向心力是**效果力**，不是一种新的基本力。它是由重力、弹力、摩擦力等一个或几个力的合力来提供的。分析受力时，绝不能多画一个向心力。
    *   **核心公式**：`Fₙ = m(v²/r) = mω²r`
- **向心加速度 (aₙ)**：
  - 定义：由向心力产生的加速度，其作用是改变速度的**方向**。
  - **方向**：始终指向圆心，与线速度方向垂直。
  - **核心公式**：`aₙ = v²/r = ω²r`

#### **2. 核心公式**
1. **线速度与角速度关系**：`v = ω r`
2. **向心加速度**：
   - `aₙ = v² / r`
   - `aₙ = ω² r`
   - `aₙ = (4π² / T²) r`
3. **向心力**：
   - `Fₙ = m aₙ = m v² / r`
   - `Fₙ = m ω² r`
   - `Fₙ = m (4π² / T²) r`

#### **3. 应用与拓展**
- **变速圆周运动**：合力不指向圆心，切向力改变速度大小。
- **一般曲线运动**：分段视为圆周运动分析（图6.2-6）。
- **实际问题**：
  - 火车转弯：外轨高于内轨，使轨道支持力与重力的合力提供水平方向的向心力（图6.4-3）。
  - 汽车过拱桥：最高点 `F_N = mg - mv²/r`（图6.4-4）。
  - **汽车过凹形路面**：在最低点，`Fₙ - mg = mv²/r`，支持力 `Fₙ > mg`（超重状态）。
  - **离心运动**：当提供的合力**小于**所需要的向心力时，物体将做远离圆心的运动（图6.4-7）。

### **第七章 万有引力与宇宙航行**
#### **核心**：万有引力定律及天体运动规律
#### **1. 开普勒三定律**
1. **轨道定律**：行星轨道为椭圆，太阳位于其中一个焦点（简化为圆轨道）。
2. **面积定律**：行星与太阳连线在等时间内扫过等面积（近日点快，远日点慢）。
3. **周期定律**：`a³ / T² = k`（`a`为半长轴）。所有行星轨道的半长轴a的三次方与公转周期T的二次方的比值是一个**常量**
*   **教师提示**：在高中阶段，通常将天体运动近似为**匀速圆周运动**来处理，此时万有引力提供向心力。

#### **2. 万有引力定律**
- **内容**：`F = G (m₁ m₂ / r²)`
- **引力常量**：`G ≈ 6.67 × 10⁻¹¹ N·m²/kg²`（卡文迪什测定，图7.2-3）。
- **重力与万有引力的关系**：在地球表面，忽略自转影响时，物体所受重力约等于地球对它的万有引力，`mg ≈ G(Mm_地/R²)`。

#### **3. 天体运动分析**
- **基本方程**：万有引力提供向心力
  - `G (M m / r²) = m v² / r`
  - `G (M m / r²) = m ω² r`
  - `G (M m / r²) = m (4π² / T²) r`
- **地球表面重力**：`mg = G (M m / R²)` → `g = G M / R²`
- **中心天体质量**：`M = (4π² r³) / (G T²)`
    *   **计算天体质量**：核心思路是**万有引力提供向心力**。
        *   `G(Mm/r²) = m(4π²/T²)r`  =>  中心天体质量 `M = 4π²r³ / (GT²)`
    *   **发现未知天体**：海王星的发现是万有引力定律预测能力的伟大证明。

#### **4. 宇宙航行**
- **宇宙速度**：
  - 第一宇宙速度（环绕）：`v₁ = √(g R) ≈ 7.9 km/s`。是人造卫星的最小发射速度和最大环绕速度。
  - 第二宇宙速度（逃逸地球）：`v₂ = √2 v₁ ≈ 11.2 km/s`。挣脱地球引力束缚。
  - 第三宇宙速度（逃逸太阳系）：`v₃ ≈ 16.7 km/s`。挣脱太阳引力束缚。
- **同步卫星**：轨道在赤道正上方，轨道高度≈36,000 km，周期=地球自转周期。

*   **5. 相对论时空观与牛顿力学的局限性**
    *   **核心结论**：时间和空间不是绝对的，而是与观察者的运动状态有关。
    *   **两大效应**：
        *   **时间延缓（钟慢效应）**：`Δt = Δτ / √(1 - v²/c²)`
        *   **长度收缩（尺缩效应）**：`l = l₀√(1 - v²/c²)`
    *   **牛顿力学适用范围**：适用于**低速（v << c）、宏观**的物体世界。

### **第八章 机械能守恒定律**
#### **核心**：功、能概念及守恒定律
#### **1. 功与功率**
- **功的定义**：`W = F l cos θ`（标量，单位 J）。功是标量，但有正负之分。
  - 正功：动力做功（`θ < 90°`）
  - 负功：阻力做功（`θ > 90°`）
- **功率**：描述做功快慢。
  - 平均功率：`P = W / t`
  - 瞬时功率：`P = F v cos θ`（汽车启动问题，图8.1-4）

#### **2. 能量**
- **动能**：`E_k = ½ m v²`
- **重力势能**：`E_p = m g h`（相对参考平面）
- **弹性势能**：与形变量和劲度系数相关（定性分析）。
- **机械能**：`E = E_k + E_p`

#### **3. 核心定律**
1. **动能定理**：
   - `W_合 = ΔE_k = E_{k2} - E_{k1}`
   - 应用：求变力做功（例：子弹穿木块）。
2. **机械能守恒定律**：
   - 条件：只有重力或弹力做功。
   - 表达式：`E_1 = E_2` 或 `ΔE_k + ΔE_p = 0`
   - 实例：单摆（图8.4-5）、弹簧振子（图8.4-9）。

#### **4. 实验验证**
- **自由落体法**：测量 `mgh` 与 `½mv²` 是否相等（图8.5-2）。
- **气垫导轨法**：减小摩擦影响（图8.5-4）。

### **重点思想与方法总结**
1. **分解思想**：
   - 曲线运动 → 直线运动（如平抛分解为水平匀速+竖直自由落体）。
2. **守恒思想**：
   - 机械能守恒（只有重力/弹力做功）。
   - 开普勒定律中的常量（`a³/T² = k`）。
3. **模型构建**：
   - 天体运动 → 匀速圆周运动（万有引力提供向心力）。
   - 变加速问题 → 动能定理（忽略过程细节）。
4. **实验方法**：
   - 频闪照相研究平抛。
   - 卡文迪什扭秤测 `G`。
   - 光电门测瞬时速度。
