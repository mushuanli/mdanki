# 第一章 直线与方程

这一章是整个解析几何的基础，核心思想是“数形结合”。我们要学会用代数的方法（方程）来研究几何图形（直线）的性质和关系。

## 一、 必须掌握的内容
本章的核心是围绕“直线”展开的。你需要掌握如何用不同的代数形式来唯一确定一条直线，并能通过这些代数形式（方程）来判断、计算两条直线之间的位置关系（平行、垂直、相交）以及相关的距离问题。

## 二、 核心定义与定理
*   定义：
    *   倾斜角 (Angle of Inclination): 一条直线向上的方向与 x 轴正方向所成的最小正角 α。它的范围是 `0 ≤ α < π`。
    *   斜率 (Slope): 倾斜角 α 的正切值，记为 `k = tan(α)` (当 `α ≠ π/2`)。当直线垂直于x轴时，斜率不存在。
    *   截距 (Intercept): 直线与坐标轴交点的坐标。在 x 轴上的截距是交点的横坐标 `a`；在 y 轴上的截距是交点的纵坐标 `b`。注意截距是一个数值，不是一个点。

*   定理/定律：
    *   两条直线平行判定定理：
        *   当两条直线斜率都存在时，`l₁ // l₂ ⇔ k₁ = k₂` (且两直线不重合)。
        *   当两条直线斜率都不存在时，它们互相平行。
    *   两条直线垂直判定定理：
        *   当两条直线斜率都存在时，`l₁ ⊥ l₂ ⇔ k₁k₂ = -1`。
        *   当一条直线斜率不存在（垂直于x轴），另一条直线斜率为0（平行于x轴）时，它们互相垂直。

## 三、 核心公式
1.  斜率公式： 经过两点 `P₁(x₁, y₁)` 和 `P₂(x₂, y₂)` 的直线斜率 `k = (y₂ - y₁) / (x₂ - x₁)` (其中 `x₁ ≠ x₂`)。
2.  直线方程的五种形式：
    *   点斜式 (Point-Slope Form): `y - y₁ = k(x - x₁)` (知道一点和斜率)。
    *   斜截式 (Slope-Intercept Form): `y = kx + b` (知道斜率和y轴截距)。
    *   两点式 (Two-Point Form): `(y - y₁) / (y₂ - y₁) = (x - x₁) / (x₂ - x₁)` (知道两点，`x₁≠x₂`, `y₁≠y₂`)。
    *   截距式 (Intercept Form): `x/a + y/b = 1` (知道x,y轴截距，`a≠0`, `b≠0`)。
    *   一般式 (General Form): `Ax + By + C = 0` (A, B不同时为0)。这是所有直线都能表示的通用形式。
3.  距离公式：
    *   两点间距离公式： `d = |P₁P₂| = √[(x₂ - x₁)² + (y₂ - y₁)²]`。
    *   点到直线的距离公式： 点 `P(x₀, y₀)` 到直线 `Ax + By + C = 0` 的距离 `d = |Ax₀ + By₀ + C| / √(A² + B²)`。
    *   平行直线间距离公式： 两条平行直线 `Ax + By + C₁ = 0` 和 `Ax + By + C₂ = 0` 之间的距离 `d = |C₁ - C₂| / √(A² + B²)`。
4.  中点坐标公式： 线段 `P₁P₂` 的中点M坐标为 `M((x₁ + x₂)/2, (y₁ + y₂)/2)`。

## 四、 重要性质与特性
*   斜率与倾斜角的关系： `k = tan(α)`。`k>0` 时，α为锐角；`k<0` 时，α为钝角；`k=0` 时，α=0。
*   直线一般式方程 `Ax + By + C = 0` 的系数与斜率、截距的关系：
    *   斜率 `k = -A/B` (B≠0)。
    *   y轴截距 `b = -C/B` (B≠0)。
    *   x轴截距 `a = -C/A` (A≠0)。
*   直线系方程： 经过两条直线 `l₁: A₁x+B₁y+C₁=0` 和 `l₂: A₂x+B₂y+C₂=0` 交点的直线系方程可表示为 `A₁x+B₁y+C₁ + λ(A₂x+B₂y+C₂)=0` (λ为参数，此直线系不包含`l₂`)。这是一个非常有用的解题技巧。

## 五、 典型题型与方法
1.  求直线的斜率、倾斜角和方程：
    *   题型： 给出点的坐标、斜率、截距等条件中的任意两项，求直线方程。
    *   方法： 根据已知条件，灵活选用点斜式、斜截式、两点式、截距式。最终结果一般要求化为一般式。要特别注意斜率不存在的垂直情况和截距为0的特殊情况。

2.  判断两直线的位置关系：
    *   题型： 给出两条直线的方程，判断它们是平行、垂直、相交还是重合。
    *   方法： 将方程化为斜截式或提取一般式中的系数，计算斜率 `k`。利用 `k₁ = k₂` 判断平行，`k₁k₂ = -1` 判断垂直。注意讨论斜率不存在的情况。对于一般式，也可以用系数关系 `A₁B₂ - A₂B₁ = 0` (平行或重合) 和 `A₁A₂ + B₁B₂ = 0` (垂直) 来判断。

3.  求交点坐标：
    *   题型： 求两条相交直线的交点坐标。
    *   方法： 联立两条直线的方程，解二元一次方程组。解出的 `(x, y)` 就是交点坐标。

4.  求解距离问题：
    *   题型： 计算点与点、点与直线、平行直线之间的距离。
    *   方法： 熟练、准确地套用对应的距离公式。计算点到直线的距离时，必须先将直线方程化为一般式 `Ax+By+C=0`。

5.  对称问题：
    *   题型： 求一个点关于某条直线的对称点；求一条直线关于某条直线的对称直线。
    *   方法： 核心是利用“中点”和“垂直”两个条件。点 `P` 和对称点 `P'` 的连线被对称轴 `l` 垂直平分。即 `PP'` 的中点在 `l` 上，且直线 `PP'` 与 `l` 垂直。

6.  与直线相关的几何综合题：
    *   题型： 求解三角形的边、中线、高线所在的直线方程；求三角形的面积等。
    *   方法： 将几何问题坐标化，综合运用斜率、方程、距离、交点等知识进行计算。例如，求高线方程就是求过一个顶点且与对边垂直的直线方程。

# 第二章 圆与方程

这一章我们学习第二种常见的曲线——圆。方法和第一章类似，继续运用坐标法，通过方程来研究圆的性质以及它和直线的位置关系。

## 一、 必须掌握的内容
本章的核心是建立圆的代数方程，并利用方程解决几何问题。你需要掌握圆的两种方程形式及其相互转化，并能够利用代数方法（解方程组）和几何方法（距离法）判断直线与圆、圆与圆的位置关系。

## 二、 核心定义与定理
*   定义：
    *   圆 (Circle): 平面内与一个定点（圆心）的距离等于一个定长（半径）的点的集合（轨迹）。
*   定理：
    *   直线与圆的位置关系判定： 圆心到直线的距离 `d` 与半径 `r` 的关系决定了直线与圆的位置关系。
        *   `d > r` ⇔ 相离 (无公共点)。
        *   `d = r` ⇔ 相切 (一个公共点)。
        *   `d < r` ⇔ 相交 (两个公共点)。
    *   圆与圆的位置关系判定： 两圆圆心距 `d` 与两圆半径 `r₁`, `r₂` 的关系决定了两圆的位置关系。
        *   `d > r₁ + r₂` ⇔ 外离。
        *   `d = r₁ + r₂` ⇔ 外切。
        *   `|r₁ - r₂| < d < r₁ + r₂` ⇔ 相交。
        *   `d = |r₁ - r₂|` ⇔ 内切。
        *   `d < |r₁ - r₂|` ⇔ 内含。

## 三、 核心公式
1.  圆的方程：
    *   标准方程 (Standard Form): `(x - a)² + (y - b)² = r²`。圆心为 `(a, b)`，半径为 `r`。
    *   一般方程 (General Form): `x² + y² + Dx + Ey + F = 0`。
        *   当 `D² + E² - 4F > 0` 时，表示一个圆。
        *   圆心为 `(-D/2, -E/2)`，半径 `r = (1/2)√(D² + E² - 4F)`。
2.  弦长公式： 直线与圆相交所得的弦长 `L = 2√(r² - d²)`，其中 `r` 是半径，`d` 是圆心到直线的距离。

## 四、 重要性质与特性
*   确定一个圆需要三个独立的几何条件（例如：三个不在同一直线上的点，圆心和半径，圆心和切线等）。
*   圆的切线垂直于过切点的半径。
*   过圆外一点作圆的两条切线，切线长相等。

## 五、 典型题型与方法
1.  求圆的方程：
    *   题型： 根据给定的圆心、半径、直径、或经过的某些点、或与某直线相切等条件求圆的方程。
    *   方法： 若已知圆心和半径，首选标准方程。若已知三点，可设一般方程，将三点坐标代入解方程组；也可以通过求两弦的中垂线的交点来确定圆心。这种方法叫“待定系数法”。

2.  判断点、直线、圆与圆的位置关系：
    *   题型： 判断点与圆、直线与圆、圆与圆的位置关系。
    *   方法：
        *   点与圆： 计算点到圆心的距离，与半径比较。
        *   直线与圆： 推荐使用几何法，即计算圆心到直线的距离 `d`，再与半径 `r` 比较。代数法（联立方程看判别式 `Δ`）计算量较大，但也是一种有效方法。
        *   圆与圆： 计算两圆的圆心距 `d`，与两圆半径 `r₁`, `r₂` 的和或差进行比较。

3.  求切线方程：
    *   题型： 求过圆上一点的切线；求过圆外一点的切线；求斜率已知的切线。
    *   方法： 核心是利用“圆心到切线的距离 `d` 等于半径 `r`”。可以设出切线方程（点斜式或斜截式），然后利用点到直线距离公式建立等式求解。

4.  弦长与弦中点问题：
    *   题型： 求直线被圆截得的弦长。
    *   方法： 利用弦长公式 `L = 2√(r² - d²)`。关键是求出圆心到直线的距离 `d`。

# 第三章 圆锥曲线与方程

这一章是高中解析几何的重点和难点，我们将在圆的基础上，学习椭圆、双曲线、抛物线这三种重要的曲线。核心方法依然是坐标法。

## 一、 必须掌握的内容
你需要牢牢掌握椭圆、双曲线、抛物线的定义、标准方程、几何性质（范围、对称性、顶点、焦点、准线、渐近线、离心率等），并能够根据这些性质解决问题。

## 二、 核心定义与定理
*   定义：
    *   椭圆 (Ellipse): 平面内到两个定点（焦点）F₁, F₂ 的距离之和为常数 `2a`（`2a > |F₁F₂|`）的点的轨迹。
    *   双曲线 (Hyperbola): 平面内到两个定点（焦点）F₁, F₂ 的距离之差的绝对值为常数 `2a`（`0 < 2a < |F₁F₂|`）的点的轨迹。
    *   抛物线 (Parabola): 平面内到一个定点（焦点）F 和一条定直线（准线）`l` 的距离相等的点的轨迹。

*   定理：
    *   圆锥曲线的统一定义：平面内到一个定点F和一条定直线l的距离之比为常数e的点的轨迹。
        *   `0 < e < 1` 时是椭圆。
        *   `e > 1` 时是双曲线。
        *   `e = 1` 时是抛物线。
        这个常数 `e` 就是离心率。

## 三、 核心公式
1.  椭圆 (焦点在x轴上为例):
    *   标准方程: `x²/a² + y²/b² = 1` (a > b > 0)。
    *   参数关系: `a² = b² + c²`。
    *   离心率: `e = c/a` (0 < e < 1)。
2.  双曲线 (焦点在x轴上为例):
    *   标准方程: `x²/a² - y²/b² = 1` (a > 0, b > 0)。
    *   参数关系: `c² = a² + b²`。
    *   离心率: `e = c/a` (e > 1)。
    *   渐近线方程: `y = ±(b/a)x`。
3.  抛物线 (开口向右为例):
    *   标准方程: `y² = 2px` (p > 0)。
    *   焦点: `F(p/2, 0)`。
    *   准线: `x = -p/2`。
    *   离心率: `e = 1`。

## 四、 重要性质与特性
*   椭圆： 有长轴、短轴，关于长轴、短轴及中心对称。
*   双曲线： 有实轴、虚轴，两条渐近线，关于实轴、虚轴及中心对称。双曲线的图形在渐近线所夹的区域内。
*   抛物线： 只有一条对称轴，一个顶点，没有中心。

## 五、 典型题型与方法
1.  求圆锥曲线的标准方程：
    *   题型： 根据给定的焦点、顶点、准线、渐近线、离心率或曲线上的点等条件求标准方程。
    *   方法： “先定位，后定量”。首先根据焦点位置判断方程类型（焦点在哪个轴上）。然后根据条件建立关于 `a, b, c` 的方程组，解出 `a², b²`，写出方程。这是最基本的题型。

2.  直线与圆锥曲线的位置关系：
    *   题型： 判断直线与圆锥曲线的交点个数；求弦长；涉及弦中点的问题。
    *   方法： 联立直线和曲线的方程，消元后得到一个一元二次方程。利用判别式 `Δ` 判断交点个数。利用韦达定理 `x₁+x₂ = -B/A`, `x₁x₂ = C/A` 结合弦长公式 `L = √[(1+k²)((x₁+x₂)² - 4x₁x₂)]` 求解。中点问题常用“点差法”或韦达定理。

3.  涉及圆锥曲线定义的题目：
    *   题型： 题目条件直接与焦点和距离有关。
    *   方法： 看到“到两定点距离和/差”或“到定点与定直线距离”等关键词，要立刻回归定义，将几何关系转化为代数关系求解。例如，求椭圆上的点到某焦点的距离，可以利用定义 `|PF₁| + |PF₂| = 2a` 来转化。

# 第四章 数列

这一章我们从几何研究转向代数研究，主要学习两种特殊的数列模型：等差数列和等比数列。

## 一、 必须掌握的内容
你需要理解数列的概念，掌握等差数列和等比数列的定义、通项公式、前n项和公式，并能运用这些公式解决问题。

## 二、 核心定义与定理
*   定义：
    *   数列 (Sequence): 按一定次序排列的一列数。
    *   等差数列 (Arithmetic Progression): 从第二项起，每一项与它的前一项的差等于同一个常数（公差 `d`）的数列。
    *   等比数列 (Geometric Progression): 从第二项起，每一项与它的前一项的比等于同一个不为零的常数（公比 `q`）的数列。

*   定理：
    *   等差中项： 若 `a, A, b` 成等差数列，则 `A = (a+b)/2`。
    *   等比中项： 若 `a, G, b` 成等比数列，则 `G² = ab`。

## 三、 核心公式
1.  等差数列：
    *   通项公式: `aₙ = a₁ + (n-1)d`。
    *   前n项和公式: `Sₙ = n(a₁ + aₙ) / 2 = na₁ + n(n-1)d / 2`。
2.  等比数列：
    *   通项公式: `aₙ = a₁qⁿ⁻¹`。
    *   前n项和公式: `Sₙ = a₁(1 - qⁿ) / (1 - q)` (q≠1)；`Sₙ = na₁` (q=1)。
3.  通项与前n项和的关系：
    *   `a₁ = S₁`。
    *   `aₙ = Sₙ - Sₙ₋₁` (n ≥ 2)。这个关系对任何数列都成立，非常重要。

## 四、 重要性质与特性
*   等差数列性质： 若 `m+n = p+q`，则 `aₘ + aₙ = aₚ + a_q`。
*   等比数列性质： 若 `m+n = p+q`，则 `aₘaₙ = aₚa_q`。
*   等差数列的通项是关于 `n` 的一次函数（或常数函数）；前n项和是关于 `n` 的二次函数（常数项为0）。
*   等比数列的通项是指数函数类型。

## 五、 典型题型与方法
1.  基本量求解：
    *   题型： 已知 `a₁, d(q), n, aₙ, Sₙ` 中的任意三个量，求另外两个量。
    *   方法： 建立方程组求解，这被称为“基本量法”。

2.  判断数列类型及求通项/求和：
    *   题型： 判断一个数列是否为等差或等比数列；求一个数列的通项公式或前n项和。
    *   方法： 利用定义，验证 `aₙ₊₁ - aₙ` 是否为常数，或 `aₙ₊₁ / aₙ` 是否为常数。求通项时，若 `Sₙ` 已知，用 `aₙ = Sₙ - Sₙ₋₁`。

3.  实际应用问题：
    *   题型： 增长率、储蓄、分期付款等问题。
    *   方法： 准确判断问题是等差模型还是等比模型，建立数学模型后利用公式求解。

4.  数学归纳法 (Mathematical Induction):
    *   题型： 证明与正整数 `n` 有关的等式或不等式成立。
    *   方法： 严格遵循两步：(1) 奠基： 验证 `n` 取第一个值（通常是1）时命题成立。(2) 归纳递推： 假设 `n=k` 时命题成立，然后利用这个假设去证明 `n=k+1` 时命题也成立。两步缺一不可。

# 第五章 导数及其应用

这一章是微积分的入门，引入了一个极其重要的概念——导数。导数是研究函数变化率的强大工具。

## 一、 必须掌握的内容
你需要理解导数的概念及其几何意义，熟练掌握基本初-等函数的导数公式和导数的运算法则，并能利用导数研究函数的单调性、极值和最值，解决优化问题。

## 二、 核心定义与定理
*   定义：
    *   导数 (Derivative): 函数 `y = f(x)` 在点 `x₀` 处的导数 `f'(x₀)` 是函数值在该点的瞬时变化率，即 `f'(x₀) = lim(Δx→0) [f(x₀ + Δx) - f(x₀)] / Δx`。
    *   导函数 (Derivative Function): `f'(x)`。
    *   极值 (Extremum): 极大值和极小值的统称。极大值是函数在某点附近一个“小山峰”的顶，极小值是“小山谷”的底。

*   定理：
    *   函数单调性与导数的关系：
        *   若在某区间上 `f'(x) > 0`，则 `f(x)` 在该区间上单调递增。
        *   若在某区间上 `f'(x) < 0`，则 `f(x)` 在该区间上单调递减。
    *   函数取到极值的必要条件： 若 `f(x)` 在 `x₀` 处可导且取到极值，则 `f'(x₀) = 0`。
    *   函数取到极值的充分条件： 在 `f'(x₀) = 0` 的前提下，若在 `x₀` 左侧 `f'(x) > 0` 且右侧 `f'(x) < 0`，则 `f(x₀)` 为极大值；若左侧 `f'(x) < 0` 且右侧 `f'(x) > 0`，则 `f(x₀)` 为极小值。

## 三、 核心公式
1.  基本初等函数求导公式：
    *   ` (C)' = 0 ` (C为常数)
    *   ` (xᵃ)' = axᵃ⁻¹ `
    *   ` (sin x)' = cos x `
    *   ` (cos x)' = -sin x `
    *   ` (eˣ)' = eˣ `
    *   ` (aˣ)' = aˣ ln a `
    *   ` (ln x)' = 1/x `
    *   ` (logₐx)' = 1/(x ln a) `
2.  导数的运算法则：
    *   ` (u ± v)' = u' ± v' `
    *   ` (uv)' = u'v + uv' `
    *   ` (u/v)' = (u'v - uv') / v² `
3.  复合函数求导法则（链式法则）： ` [f(g(x))]' = f'(g(x)) · g'(x) `。本书中主要应用在 `y = f(ax+b)` 型，其导数为 `y' = a · f'(ax+b)`。

## 四、 重要性质与特性
*   导数的几何意义： 函数 `y=f(x)` 在点 `x₀` 处的导数 `f'(x₀)` 是曲线在该点 `(x₀, f(x₀))` 处切线的斜率。
*   导数的物理意义： 如果位移是时间的函数 `s(t)`，那么 `s'(t)` 就是瞬时速度 `v(t)`；如果速度是时间的函数 `v(t)`，那么 `v'(t)` 就是瞬时加速度 `a(t)`。

## 五、 典型题型与方法
1.  求函数的导数：
    *   题型： 求给定函数的导函数，或在某一点的导数值。
    *   方法： 熟练运用求导公式和运算法则，对于复杂函数要能准确拆解，分步求导。

2.  求曲线的切线方程：
    *   题型： 求曲线在某点处的切线方程。
    *   方法： 分两步。(1) 求导数，并将该点的横坐标代入，得到切线斜率 `k = f'(x₀)`。(2) 利用点斜式 `y - y₀ = k(x - x₀)` 写出切线方程。

3.  利用导数研究函数的单调性与极值：
    *   题型： 求函数的单调区间、极值点和极值。
    *   方法： (1) 求定义域。(2) 求导函数 `f'(x)`。(3) 令 `f'(x) = 0`，解出方程的根（驻点）。(4) 用这些根将定义域划分为若干区间，列表分析各区间 `f'(x)` 的符号，从而确定 `f(x)` 的单调性，并根据符号变化情况判断极值。

4.  求函数在闭区间上的最值：
    *   题型： 求函数 `f(x)` 在区间 `[a, b]` 上的最大值和最小值。
    *   方法： (1) 求出 `f(x)` 在开区间 `(a, b)` 内的所有极值。(2) 计算这些极值和区间两个端点的函数值 `f(a)`, `f(b)`。(3) 比较所有这些值，其中最大的就是最大值，最小的就是最小值。

5.  导数的应用（优化问题）：
    *   题型： 解决实际生活中的最优化问题，如利润最大、用料最省、面积最大等。
    *   方法： (1) 建立目标函数模型和定义域。(2) 利用导数求函数的最值。(3) 回归实际问题作答。
